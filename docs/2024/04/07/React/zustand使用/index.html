一、简介

Zustand是适用于React的一个状态管理工具
Zustand 是由 Jotai 和 React springs 的开发人员构建的快速且可扩展的状态管理解决方案。
Zustand是状态管理器和观察者模式的代表。

二、优点

  • 使用简单、代码简洁，消除provides、actions、dispatch等
  • 只在state状态变化时重新渲染组件
  • 体积小，压缩后不到1KB
  • Zustand使用 hooks 来管理状态无需样板代码
