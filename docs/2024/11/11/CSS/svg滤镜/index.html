原理

feTurbulence创建了一个位移图，然后通过feDisplacementMap应用位移效果。scale设置为20，表示位移的大小，而xChannelSelector和yChannelSelector分别设置为R和G，表示使用位移图的红色和绿色通道来决定x轴和y轴的位移。
