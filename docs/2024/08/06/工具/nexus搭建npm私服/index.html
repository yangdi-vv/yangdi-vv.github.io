一、nexus介绍

JetBrains开发的仓库管理工具，它提供了Maven和Sonatype NXG（之前称为Ossium）的集中式存储库管理。

二、背景

  • 网络稳定性和业务稳定性

    私有仓库由公司自己维护，能够快速响应并解决访问或服务宕机等问题,同时避免线上npm包被作者删除

  • 扩展性和配置多样性

    私有仓库提供了更多定制化的配置选项，可以根据公司具体需求灵活调整,同时私有仓库支持对npm包进行拓展或者结合CICD流程。

  • 代码私密性

    公司内部的业务代码不对外开放，对代码更严格的审查，避免恶意代码攻击。
