find指令的使用

发表于 2024-10-09
  • One Piece
首页 linux

介绍

全称 File IN Databank，是一个强大的文件搜索命令，可以根据各种标准如文件名、大小、修改日期等在系统中搜索文件和目录。
find 这个命令适用于所有Linux版本，包括但不限于 Ubuntu、Debian、Fedora、Red Hat、CentOS 等。由于 find 是预安装在所有Linux发行版上的，所以大部分时候并不需要进行额外的安装。

使用

参数 说明
-name 搜索文件名
-size 根据文件大小搜索文件
-type 根据类型搜索文件，如 -type d 为搜索目录
-mtime [n] 根据修改时间搜索文件，n为天数
-exec 将查找的文件执行指定的命令

  • 查找名称为filename的文件

  • 搜索所有大于1MB的文件

  • 查找当前目录及子目录下所有的 .txt 文件

  • 在/home目录中查找以.log结尾的文件

  • 查找/home目录中用户所有者为root，并且文件大小超过10M的文件

  • 在当前目录查找更改时间在5天之内的文件

  • 找出所有的空目录

  • 查找并删除所有的 .tmp 文件

  • 查找并打印所有以 .sh 结尾的文件，并显示其文件大小和最后修改时间

  • 查找当前目录及子目录中文件权限为755的文件

  • 在/home目录中查找拥有者为root的文件

  • 查找/home目录中，更改时间在10天前，15天内的文件

  • 查找/etc目录下，更改时间超过10天，并且后缀为’.log’的文件

  • 查找/var/log目录下，10天内被访问过，并且后缀为’.log’的文件
