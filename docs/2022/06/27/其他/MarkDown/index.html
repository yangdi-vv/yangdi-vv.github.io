MarkDown 基本语法

一：MarkDown 是什么

MarkDown 是一种纯文本标记语言，按照MD字符使用规范，使普通的文本在支持MD预览的工具中显示出特定的排版格式。

二：MarkDown 优点&缺点

优点：操作简单、编写方便。
缺点：需要学习使用语法，同时对MD支持的工具才可以进行特殊排版的展示及格式转换，否则只是纯文本。
