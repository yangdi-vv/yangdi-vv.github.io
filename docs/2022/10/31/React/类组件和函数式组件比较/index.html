区别

区别 函数组件 类组件
生命周期 无 有
this 无 有
state 无 有
改变state React Hooks:useState this.setState()
性能 高 低 （需要实例化）
  • 主要区别

    函数式组件捕捉的是渲染时传入的值，props及useState都可以可以使用setTimeout来体现这个问题，类函数和函数式逐渐区别比较大的一点
