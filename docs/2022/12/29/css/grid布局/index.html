一、grid布局

1、基本概念

Grid 网格布局，将网页划分为一个个网格，可以任意组合不同的网格，对项目进行不同的排列顺序，flex是一维布局，只能针对项目的轴线设置项目放置的相对位置，而grid布局是将容器划分成行和列，产生的单元格，然后指定项目所在的位置，可以把它看成是一个二维布局。

  • 行
  • 列
  • 间隔
  • 区域

二、容器属性

1、grid-template-columns 和 grid-template-rows:分别设置属性的列宽和行高 补充：





2、grid-gap 设置行间隙和列间隙大小

3、grid-template-areas 定义多个区域及区域的排列方式（区域客服用出现多次）

4、grid-auto-flow: row 自动布局算法，在一行内能排列下的情况下，依照先行后列（默认）的原则依次布局排列。也可以设置先列后行的写法，即：grid-auto-flow: column





5、align-items: start 设置单元格内的项目的垂直（上中下）位置




6、justify-items 设置单元格内的项目的水平（左中右）位置；align-items设置单元格（项目）内容的垂直（上中下）位置；place-items = align-items +justify-items 这俩个属性的写法完全相同




7、place-items: align-items + justify-items

8、justify-content、align-content 设置容器内所有内容的垂直和水平方向




9、place-content是justify-content、align-content 合并

三、项目属性

项目的属性是针对一个大的div中的具体的内容设置的元素（也就是给大箱子中的小箱子设置样式）。一共6有个属性，分别是：order、flex-grow、flex-shrink、flex-basis、flex、align-self。

1、grid-row-start: 属性指定哪一行开始显示网格元素。

2、grid-row-end 属性指定哪一行停止显示网格元素，或设置跨越多少行。

3、grid-column-start：指定网格元素列的开始位置

4、grid-column-end：指定网格元素列的结束位置
