flex布局

一、概念

Flex 是 Flexible Box 的缩写，意为”弹性布局”，用来为盒状模型提供最大的灵活性。任何一个容器都可以指定为 Flex 布局。采用 Flex 布局的元素，称为 Flex 容器，它的所有子元素自动成为容器成员，称为 Flex 项目。
注： 设为 Flex布局以后，子元素的float、clear和vertical-align属性将失效。

二、容器属性

1、flex-direction: row(默认) - 项目主轴方向

  • 1、flex-direction: row(默认) - 项目主轴方向

2、flex-warp:nowrap （默认）当一条轴线上排列不下，是否换行


3、flex-flow: row nowrap;(默认) 相当于合并flex-flow: flex-direction flex-warp;


4、justify-content: flex-start(默认) 设置主轴上的排列方式





5、align-items: flex-start




6、align-content: flex-start （默认） 设置侧轴上的对齐方式（多行）






三、项目属性

项目的属性是针对一个大的div中的具体的内容设置的元素（也就是给大箱子中的小箱子设置样式）。一共6有个属性，分别是：order、flex-grow、flex-shrink、flex-basis、flex、align-self。

1、order: 容器中项目的排列顺序，数值越小项目越靠前,默认为0，如下order=1

2、flex-grow: 属性定义项目存在剩余空间时，项目增加剩余空间比例的宽度，默认值是flex-grow: 0。即使当宽度固定，子项目也会被放大，如下。

3、flex-shrink：超出容器大小时，项目宽高的缩小比例，默认为flex-shrink: 1，值越大，按比例缩小的越多。

4、flex-basis：在分配多余的空间之前，根据项目占据的主轴空间去计算在主轴上是否有多余的空间。默认值flex-basis: | auto; 即，项目本身的大小，我们设置项目元素的width的大小。

5、flex-grow + flex-shrink + flex-basis = flex。定义子项目分配剩余空间，用flex来表述占多少份。

7、align-self: 默认值为align-self:auto,继承父元素设置的align-items的值。控制子项目在侧轴的对其方式。
