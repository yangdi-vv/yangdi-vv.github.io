二：jmeter

JMeter 最初被设计用于 Web 应用测试，但后来扩展到了其他测试领域，可用于测试静态和动态资源，如静态文件、Java 小服务程序、CGI 脚本、Java 对象、数据库和 FTP 服务器等等。

  • JAVA 环境运行
  • 支持跨平台 并且支持多协议

三：ant

Apache Ant是目前事实上的Java应用的标准build脚本工具，可以将软件编译、测试、部署等步骤联系在一起加以自动化，可以使用Task来对Ant进行灵活扩展。

  • Ant是由一个内置任务和可选任务组成的

四、docker

1、java + ant + jmeter 制作基础镜像 （无需操作，已制作）

2、下载coding仓库，并使用以上基础镜像制作测试镜像 （需手动操作）

3、启动并进入容器，执行导出报告指令 （需手动操作）
